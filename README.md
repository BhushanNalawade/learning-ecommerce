# learning-ecommerce
- a full-stack project to build a classic e-commerce with products, cart and orders

## Dependencies
- Node 20 LTS for backend
- React 18 for frontend

## Getting Started
- install node 20 LTS
- clone project
- install mongodb

- have two terminals open to project root 

### Backend
- rename `.env.example` to `.env`
- `cd backend`
- `npm install`
- `npm run dev`

### Frontend
- `cd frontend`
- `npm install`
- `npm start`
